## AI projects I am building

- **CeroClawd CLI** - Open-source CLI for running multi-agent workflows locally with tool calling, terminal execution, and local models through Ollama. Public repo: [gonzalonicolasr/ceroclawd](https://github.com/gonzalonicolasr/ceroclawd). Site: [cli.ceroclawd.com](https://cli.ceroclawd.com)
- **ceroclawd.com** - Browser-based platform where freelancers can create AI teams, assign work, monitor execution, and manage projects without leaving the web app.
- **OfficeClaw** - Real-time operations dashboard that visualizes agent activity, workflow state, logs, and team coordination.
- **CeroSpace** - Desktop workspace for local agent execution with multi-terminal layout, flow visualization, and a local server powered by `node-pty`, WebSocket, and Express.
- **Memoria + Cortex** - Persistent memory layer for coding agents with SQLite storage, MCP tooling, project context, shared memory, and workspace-level knowledge reuse.
- **CeroClaw Serve** - Internal execution engine that runs the CLI in serve mode and streams agent activity to the web platform over HTTP + SSE.
- **Cortex Video** - Remotion-based pipeline used to create product demos and launch assets for the ecosystem.
